# Практические работы по веб-разработке

## 📚 О репозитории

В этом репозитории хранятся практические работы, выполненные в рамках курса по веб-разработке.

| Работа | Описание |
|--------|----------|
| **Практическая работа №1** | Основы работы с Git и GitHub. Создание локального и удалённого репозитория, коммиты, ветки, отправка изменений |
| **Практическая работа №2 (Windows)** | Установка и настройка Open Server Panel. Создание доменов, настройка виртуальных хостов |
| **Практическая работа №2 (Linux)** | Настройка локального веб-сервера на Linux (Apache, hosts, виртуальные хосты) |

---

## 🛠️ Технологии

- **Git / GitHub** — система контроля версий
- **HTML / PHP** — основы веб-разработки
- **Open Server Panel** — локальный веб-сервер для Windows
- **Apache** — веб-сервер
